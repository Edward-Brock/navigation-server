# Navigation Server

✨ Navigation 开源导航，前端使用 Vue.js / Vite 进行构建，服务端使用 Nest.js 进行构建。

前端代码请移步至此 [navigation](https://github.com/Edward-Brock/navigation)

## 使用技术

### 服务端

- [Nest.js](https://nestjs.com/)

## 项目安装及运行

### 项目安装

```bash
npm install
```

## 项目运行

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
